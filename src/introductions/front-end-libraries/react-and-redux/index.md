## React 和 Redux 挑战的介绍

这个系列的挑战介绍怎样在 React 里使用 Redux。

在一个 React Redux 应用里，创建一个 Redux store 管理 app 的所有状态。React 组件仅订阅 store 中与其角色相关的数据。接着，直接从 React 组件分发 action，触发 store 更新。
